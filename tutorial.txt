GIT e GITHUB

Comandos básicos do git:
git config --global user.name "Seu nome"
git config --global user.email "seuemail@email.com"
git clone url_do_repositório (clona um repositorio)
git status (verificar o status do repositorio local)
git add . (adiciona ao repositorio local)
git commit -m "tutorial" (documentar as alterações)
git push (enviar ao repositorio)
git pull (sincronizar as informações do repositorio no Pc)
